# Sraventix Technologies LLP

Official website for Sraventix Technologies LLP. Built with Next.js.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

## Development

Run the development server:

```bash
npm run dev
```

## Build

Build the production site:

```bash
npm run build
```

## Run

Start the production server (after build):

```bash
npm start
```
